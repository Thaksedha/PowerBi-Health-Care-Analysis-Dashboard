The Healthcare Analysis Dashboard is an interactive Power BI project designed to analyze healthcare data and deliver actionable insights for better decision-making. This dashboard provides a comprehensive view of patient details, hospital performance, treatment trends, and key healthcare metrics.

🎯 Objectives
Analyze patient demographics and medical data
Monitor hospital performance and service efficiency
Identify trends in diseases and treatments
Support data-driven healthcare decisions
🛠️ Tools & Technologies
Power BI Desktop
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)
Data Modeling
Interactive Visualizations
📂 Dataset Description

The dataset includes healthcare-related information such as:

Patient ID & Demographics (Age, Gender)
Disease / Diagnosis Details
Admission & Discharge Dates
Treatment Information
Doctor / Department Details
Billing / Cost Data
📊 Dashboard Features
🔹 Patient Analysis
Patient distribution by age and gender
Admission trends over time
🔹 Disease & Treatment Insights
Most common diseases
Treatment success and frequency
🔹 Hospital Performance
Department-wise performance
Doctor efficiency metrics
🔹 Financial Analysis
Treatment costs and billing trends
Revenue insights
📈 Key Insights
Identifies high-frequency diseases and treatment patterns
Highlights patient admission trends and peak periods
Provides cost analysis for better financial planning
Helps improve hospital resource allocation
